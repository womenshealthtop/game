About Memory Test

The Memory Test game is designed to challenge and improve your brain's memory capabilities. With every level, the difficulty increases, pushing your focus and recall abilities to the limit!

How to Play

Follow these simple steps to play the Memory Test:

  • At the start of each level, some tiles will light up in yellow for 2 seconds.
  • Memorize the position of the yellow tiles within the time limit.
  • Click on the tiles that were yellow to progress to the next level.
  • Each level introduces more tiles to remember, increasing the challenge.

Scoring System

Your score in Memory Test depends on:

  • The number of levels you successfully complete.
  • Your speed in identifying the correct tiles.
  • Bonus points for completing levels without mistakes!

Challenge yourself and aim for the highest score!
